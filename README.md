# 🍽️ API REST Restaurante

## 📌 Descripción
Este proyecto es una API REST desarrollada con Node.js, Express, Prisma y PostgreSQL. Permite gestionar usuarios, mesas y reservaciones en un restaurante, con autenticación JWT y control de roles (cliente y administrador).

---

## 🚀 Tecnologías utilizadas
- Node.js
- Express
- Prisma ORM
- PostgreSQL
- JWT (JSON Web Token)
- bcrypt
- Swagger

---

## 🔐 Autenticación y roles
La API usa autenticación con JWT.

### Roles disponibles:
- cliente
- admin
